# AI Customer Support Agent
A multi-agent AI system built using Google ADK that creates support tickets, analyzes user issues, and provides automated responses.

## Features
- Multi-agent architecture (Planner, Worker, Evaluator)
- Automated ticket generation
- Session memory
- Modular ADK-based structure
- Demo via run_demo.py

## How to Run
1. Install dependencies:
```
pip install -r requirements.txt
```
2. Run:
```
python run_demo.py
```
